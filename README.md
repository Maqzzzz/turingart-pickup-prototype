# TuringArt 拍照单精修幻灯片原型

这是一个可直接在浏览器运行的静态网页原型，用来演示“图库页 / 拍照单精修页 → Mac 照片式幻灯片 → 全屏选片”的完整体验。

## 运行方式

发布成网页链接：

将本项目上传到 Netlify、Vercel 或 GitHub Pages 这类静态托管平台即可。详细步骤见 `DEPLOY.md`。

最简单的分享方式：

1. 将整个文件夹压缩后发给对方。
2. 对方解压后，直接双击 `index.html` 打开。
3. 在“图库”或“精修”页点击“导入文件”选择一批本地照片/视频，或点击“导入文件夹”选择一个本地文件夹。

使用本地预览服务器时，音频和文件选择体验更稳定：

```bash
python3 -m http.server 4173
```

然后打开：

```text
http://127.0.0.1:4173/
```

## 替换本地文件

在“图库”或“精修”页点击“导入文件”，可以选择一批本地照片/视频；点击“导入文件夹”，可以选择任意包含照片/视频的本地文件夹。页面只在当前浏览器中读取本地文件，不会上传到服务器，也不会保存到云端。

## 主要功能

- 图库页与拍照单精修页可以相互切换，二者不显示侧边栏。
- 顶部支持创建和切换多个拍照单项目。
- 图库只显示照片缩略图，不单独放大一张照片，也不展示照片名字。
- 收藏和回收站筛选只作用于当前拍照单项目。
- 图库页选中照片后，按 `Space` 进入选片。
- 所有大图、缩略图和图库卡片都使用完整展示，不裁切铺满；选片和幻灯片按长边撑满可用屏幕。
- 精修页新增“播放幻灯片”按钮，点击后进入全屏幻灯片播放。
- 幻灯片页参考 Mac“照片”应用的播放体验，支持切换音乐、选择本地音乐、设置转场样式、调整播放速度。
- 幻灯片支持封面标题，默认“兰兰宝贝顺利平安”，可在播放设置中修改。
- 幻灯片播放时可按 `← / →` 切换照片。
- 幻灯片播放时按 `ESC` 退出播放，按 `Enter` 进入选片。
- 选片页照片全屏完整显示，支持 `← / →` 切换。
- 选片页可点击爱心收藏/取消收藏照片。
- 选片页可点击删除按钮，或按 `D` / `Delete` / `Backspace` 将照片放入回收站。
- 支持 `⌘Z` 或 `Ctrl+Z` 撤销收藏、删除和旋转操作。
- 支持按 `L / R` 对照片进行左右旋转。
- 选片模式默认展开侧边栏，按 `S` 可收起或展开；侧边栏内可切换拍照单项目。

## 素材

- 内置照片使用 `assets/child-studio-photo.svg`、`assets/home-baby-photo.svg` 和一个内嵌占位样片。
- 内置音乐位于 `assets/music/`，默认使用 `段奥娟 - 陪我长大.mp3`。
